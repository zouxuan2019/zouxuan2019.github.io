Five Steps Machine Learning Process

Share

Most commonly, the steps in using the Scikit-Learn estimator API are as follows

  • Choose a class of model by importing the appropriate estimator class from Scikit-Learn.
    from sklearn.linear_model import LinearRegression

  • Choose Model hyperparameters by instantiating this class with desired values.
    model= LinearRegression()

  • Arrange data into a features matrix and target vector.
    x=df['RM'].values.reshape(-1,1)
    y=df['MEDV'].values

  • Fit the model to your data by calling the fit() method of the model instance.
    model.fit(x,y)

  • Apply the Model to new data:
    For supervised learning, often we predict labels for unknown data using the predict() method.
    For unsupervised learning, we often transform or infer properties of the data using the transform() or predict() method.
    model.predict(np.array([7]).reshape(1,-1))

非暴力沟通

Share

生气的理由 - 需求没有被满足

  • 你生气别人骂你时，是因为你需要尊重
  • 你生气别人拿你东西，是因为你想保护自己的财产
  • 你生气孩子不听话，是因为你希望把孩子培养好
  • 你生气朋友欺骗你，是因为你需要情感归属

正确的做法：提出自己的需求，和对方商量解决办法

  • 当母亲看到儿子把脏衣服乱扔
    • 暴力沟通：常见的反应是：你怎么总是这么邋遢，你就不能放到洗衣机里吗？
    • 非暴力沟通：我看到沙发上有两只脏袜子，我不太高兴，因为我看重整洁。你是否愿意把袜子放进洗衣机？

非暴力沟通的四个步骤

  • 观察：不带评论的观察，描述行为而非攻击人格

    • 从事实入手
    • 不能言过其实
    • 一次只说一件事
    • 不要推测意图
    • 不要随意评价

  • 感受 ： 情绪源于自身

    • 遇到不满意的事
      • 分析自己的想法：“还有没有别的解释？”
      • 关注自身：“我的什么需要没有被满足？”“ta 的行为违背了我的什么价值观？”
      • 第一人称表述：“我感到。。。，因为我。。。“ 比如，我感到生气而非你惹怒我了。我有点失望而非你让我失望

  • 需要

    • 从正面说：告诉 ta 需要做什么而不是不做什么
    • 比如妻子对丈夫说：我不想让你去参加球赛，丈夫可能会感到莫名其妙。妻子应该直接说出来：我希望你周末能陪陪我和孩子
    • 如果你对家人说：不要再喝酒了。估计很难起到作用。你可以这样说：你能否告诉我喝酒满足了你的什么需要？有没有其他方式可以满足那些需要？

  • 请求 ： 提出明确的请求

    • 改变做法：比如进我房间请敲门。而不是改变性格：请考虑周到些
    • 请求不是命令：利用恐惧和内疚来实现自己的目的，那就不是请求，而是命令。
      • 恐惧：你再这样我生气了。
      • 内疚： 如果你真的爱我，就应该陪着我。
    • 请求和命令的区别在于：你有没有考虑别人的感受。
      • 提出命令的人，如果听到别人的拒绝，会感到生气，想要指责对方。
      • 提出请求的人，如果被别人拒绝，他会考虑对方有什么困难。
