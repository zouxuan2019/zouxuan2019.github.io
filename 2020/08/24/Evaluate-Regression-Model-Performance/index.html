Split data into training data and test data

Method 1: Residual Analysis

Method 2: Mean Squared Error (MSE), The lower the number the better

  • MSE=(SUM(y_real-y_predict)^2)/n ———-n is the number of y values
  • The average value of the Sums of Squared Error cost function
  • Useful for comparing different regression models
  • For tuning parameters via a grid search and corss-validation

Method 3: Coefficient of Determination R^2, The higher the number the better

  • R^2 = 1- SSE/SST
  • SSE: Sum of squared errors
  • SST: Total sum of squares
