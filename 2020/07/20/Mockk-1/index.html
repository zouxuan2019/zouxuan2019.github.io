MockK - 1

Share

Dependency

  • Maven
  • Gradle

Integrate with Junit 5

  • Add @ExtendWith(MockKExtension::class) on top of your test class

Annoations

  • @MockK: Creates mocks that are strict, so you need to provide some behaviour.
  • @MockK(relaxUnitFun = true): Make Unit returning functions to be relaxed
  • @RelaxedMockK: A relaxed mock is the mock that returns some simple value for all functions. This allows to skip specifying behavior for each case, while still allowing to stub things you need
  • @SpyK: Spies allow to mix mocks and real objects.
    Note: the spy object is a copy of a passed object.
  • @InjectMockKs: @InjectMockKs by default is injecting only lateinit vars or vars that are not assigned. To change this, use overrideValues = true. This would assign the value even if it is already somehow initialized. To inject vals, use injectImmutable = true. For a shorter notation use @OverrideMockKs which does the same as @InjectMockKs by default, but turns these two flags on.
  • Injection first tries to match properties by name, then by class or superclass.
  • Properties are injected even if private is applied. Constructors for injection are selected from the biggest number of arguments to lowest.
